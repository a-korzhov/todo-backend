## Todo App REST API 
### Practicing project

### Technologies
- Spring Boot 2 LATEST
- Spring Security
- Spring Data JPA
- JavaMail API
- Lombok & Mapstruct
- PostgreSQL
- Flyway Migrations
- Passay
- Swagger UI

### Features
- Create user with email verification
- Authorize user
- Forgot password logic by email
- Create/Delete tasks
- List tasks
- Update user data
- Upload avatar
